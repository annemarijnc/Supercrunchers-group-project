Hallo hallo, dit is de git repo voor supercrunchers

Voor degene die de git commands nog ff op een rijtje willen hebben:

ALTIJD VOORDAT JE ZELF IETS VERANDERT:
- git pull


Daarmee update je je eigen files met de versie die op git staat. Als je eerst zelf gaat editen en daarna pas pullt kan het zijn dat er een clash is en dat is soms een beetje kut om te fixen.

Als je je eigen edits will ‘uploaden’, gebruik dan:
- git add filename
- git commit -m “type hier je bericht"
- git push
